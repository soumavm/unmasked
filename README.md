# unmasked

All-in-one Covid assistant for smaller businesses.

Includes: - Flutter Android App which allows administrator to easily contact trace individuals in contact with a Covid patient.
          - Computer Vision machine learning algorithm which identifies whether someone entering is wearing a mask or not.
          - Google Firebase backend support which logs any information provided on the app (who enters and leaves the building).
          - Automatic email updates for any who may be in danger of contacting covid from a coworker.
